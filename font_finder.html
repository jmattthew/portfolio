Better Font Finder

Background

This product allows people explore, filter, and select fonts based on their visual attributes. I conceived, designed, and coded the tool as a personal project.

The core idea of the project is to create a user interface that doesn't require any explanation, doesn't use any font terminology, and encourages visual exploration. In that vein, I encourage you to try it now.
